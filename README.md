# Currency-Convertor-React

A simple and intuitive currency converter built with React.js. This application allows users to convert between different currencies using real-time exchange rates.

## Features
- Convert between multiple currencies
- Real-time exchange rates fetched from an API
- User-friendly interface with responsive design
- Option to swap currencies with a single click
